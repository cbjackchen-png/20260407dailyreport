台股盤前展望（4/7清明後復市）

清明假期後台股復市，美股四連漲提供正面背景，但地緣政治仍是首要變數。台股今日（4/7）清明假期後正式復市，4/2收盤32,572.43點，已跌破部分短期均線。美股假期期間連漲四日，費半指數五日漲幅達10.83%，記憶體與AI供應鏈題材持續發酵，加上鴻海3月營收年增45.57%創同期新高等台廠業績訊號正面，外部環境對台股初開盤提供一定支撐。然而，川普對伊朗的最後通牒截止時間正逢今日美東時間20時（台灣時間4/8凌晨8時），中東情勢在今日盤中可能出現劇烈變化，地緣政治仍是台股當日走勢的主要不確定因素。

技術面：守穩季線為短線多頭首要任務。台股在3月的震盪整理後，目前位置需留意季線（約32,000至32,500點附近）是否能有效支撐。若美伊談判在今日出現正面進展，外資回補意願有機會提升，指數挑戰月線的條件將逐步成形；若談判再度破局或軍事行動升溫，油價反彈帶來的通膨壓力將再次壓制風險資產，台股向下測試支撐的風險上升。

操作建議：聚焦業績能見度高的AI供應鏈，控制倉位留意今日川普截止期限動向。在事件風險集中的背景下，建議以具備實質業績支撐的標的為核心——鴻海AI機櫃、記憶體（HBM/DRAM漲價週期）、矽光子與光通訊，以及博通TPU長約帶動的先進封裝需求，均屬於有明確訂單可見度的配置方向。短期倉位宜保持相對靈活，今日下午至傍晚密切追蹤中東談判的最新動態。

分析師洞察：

美股連漲四日，費半本週五日漲幅達10.83%，創自1月以來最長連漲紀錄。週一（4/6）美股在極高的地緣政治不確定性下維持上漲，標普500與那斯達克連漲四個交易日，為自1月以來最長連漲紀錄。值得注意的是，盤中因川普再度升高對伊朗的強硬威脅，三大指數一度明顯回落，但尾盤因市場寄望最後關頭達成停火，多頭資金持續入場，最終全面收紅。費城半導體指數本週五日漲幅達10.83%，遠優於大盤，顯示市場資金在等待地緣政治明朗化的過程中，優先集中配置於半導體與AI供應鏈。

板塊結構：通訊服務領漲，存儲題材引領半導體反彈。S&P 500中，通訊服務類股當日漲幅最大，而公用事業類股表現最弱，顯示資金輪動偏向成長而非防禦。存儲族群成為最受矚目的亮點：希捷（+5.58%）創歷史新高，摩根士丹利將其列入首選股名單；美光（+3.15%）、閃迪（+3.28%）、西部數據（+3.11%）同步走強；三星同日宣布Q2 DRAM再漲三成，為記憶體超級週期的持續性提供正面驗證。大型科技股走勢分歧，Apple（+1.15%）、Amazon（+1.44%）、Alphabet（+1.09%）表現較強，微軟（-0.16%）、Meta（-0.25%）、博通（-0.04%）則接近持平或小跌。

美股中期仍承壓，財報季開啟為關鍵驗證窗口。從月線與年初至今的累積跌幅來看，標普500月線跌幅仍達3.20%、那斯達克跌5.36%，中期回檔壓力尚未完全消化。高盛指出「快錢」的拋售正在減弱，預計未來一個月該群體可能淨買入約550億美元（含200億美元美股），為市場提供潛在的技術性支撐。資深策略師Yardeni則指出，科技股相對高點已回調至具吸引力的長期布局水位。本週財報季正式展開，企業獲利能否支撐目前的估值水平，是後續走勢的核心觀察點。

Broadcom (AVGO) 展望正向

至2031年TPU長約博通宣布與谷歌達成開發和供應張量處理單元（TPU）的長期協議，負責為谷歌未來幾代TPU提供晶片開發與供應，並向谷歌下一代AI機架供應網絡及其他元件，協議期最長持續至2031年。此為博通迄今已知最長期的AI晶片供應承諾，確立其在谷歌AI算力架構中的核心地位。
Anthropic擴大算力合作博通、谷歌與Anthropic三方擴大現有戰略合作，自2027年起Anthropic將通過博通獲得約3.5吉瓦的算力資源。這一安排同時確保了博通在AI模型訓練與推理基礎設施的持續需求，為其定制AI晶片（XPU）業務提供明確的中長期訂單能見度。
網絡與電源供應鏈同步受惠博通除TPU本體外，亦負責谷歌AI機架的網絡元件供應，涵蓋乙太網交換晶片等關鍵組件。隨著AI資料中心的規模持續擴大，博通在網絡連接端的市場份額與議價能力同步提升。
技術面當日幾乎持平（-0.04%），長約消息屬中長期正面催化劑；後續需觀察市場是否在財報季消化此一題材後形成更明確的上行驅動。

存儲族群（STX / MU / WDC） 展望正向

希捷創歷史新高希捷科技（STX）當日大漲5.58%，收創歷史新高，摩根士丹利將其列入首選股名單，認為AI驅動的企業儲存需求正在加速復甦，且希捷作為HDD市場龍頭在此週期中具備相對獨特的受益地位。閃迪（+3.28%）、美光（+3.15%）、西部數據（+3.11%）同步走強，存儲板塊形成共振。
三星DRAM Q2再漲三成三星電子擬於Q2將DRAM價格再度調漲約30%，為記憶體超級漲價週期的持續性提供最直接的驗證。在HBM4需求激增、AI訓練對DRAM頻寬需求不斷提升的背景下，三星此舉亦為美光、南亞科、華邦電等同業帶來跟進調漲的市場預期。
SK海力士擬赴美ADRSK海力士計劃今年發行美國存託憑證（ADR），規模最高可達100億美元，使美股投資人擁有更多直接參與AI記憶體成長的管道。此舉雖短期對美光造成一定的資金競爭壓力，但長期而言反映出AI記憶體需求的市場信心，也進一步確認了記憶體超級週期的吸金效應。
技術面美光連日反彈（+3.15%），從五連跌後展開修復；希捷突破歷史高點，量能配合，技術面轉強。

資料來源: Tradingview.com

Oracle (ORCL) 展望正向

新任CFO強化財務紀律甲骨文宣布任命前施耐德電機CFO Hilary Maxson為新任財務長，負責在公司大舉投資AI與雲端基礎設施之際強化財務紀律與資本配置效率。此人事異動反映管理層對當前AI投入規模進行更嚴格的財務管控，有助改善市場對其長期資本回報的預期。
AI雲端基礎設施加速擴張甲骨文持續深化在AI雲端基礎設施的布局，其OCI（Oracle Cloud Infrastructure）在AI工作負載領域的市場滲透率持續提升，並與多家超大規模雲端業者保持合作關係。財報優異並大幅上修展望的紀錄，使市場對其AI轉型路徑維持較高的信心程度。
Anthropic策略合作擴大博通與谷歌擴大對Anthropic合作的同時，甲骨文亦透過其雲端平台積極布局AI模型推理服務。Anthropic自2027年起將通過博通獲取3.5吉瓦算力，而OCI作為底層基礎設施之一，有望在AI推理規模化過程中持續受惠。
技術面近期隨大盤整理，均線結構仍維持偏多排列，新任CFO消息有助穩定估值預期，財報季業績驗證為下一關鍵節點。

Tesla (TSLA) 中性觀察

業績表現面臨下滑風險摩根大通分析師Ryan Brinkman表示，特斯拉投資者應謹慎行事，因為該公司面臨業績表現進一步下滑的風險，當日特斯拉收跌2.15%，為美股科技五大巨頭中表現最弱的個股。
NHTSA調查結束為小利多美國公路安全管理局（NHTSA）結束對特斯拉「真．智慧召喚」功能的調查（涵蓋約259萬輛車），原因是相關事故多屬低速輕微財損，無傷亡案例，排除了潛在的大規模強制召回風險，為特斯拉帶來一定的法規不確定性解除。
Terafab計畫進展馬斯克宣布七日內公布全球最大晶片廠「Terafab」計畫，與台積電不同的是Terafab將採用Tesla專有製程技術。雖為遠期題材，但若實際落地，可能對未來Tesla AI算力自主化佈局具有深遠影響。
技術面當日-2.15%，為五大巨頭中最弱；技術面壓力猶在，短線可觀察關鍵支撐能否守穩，財報季業績驗證為中期關鍵。
